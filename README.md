# Confab

A macOS menubar app for spoken conversations with a room of AI advisors — they
talk to you and to each other, out loud.

**Confab:** <https://confab.logicaleap.com>

This repository hosts Confab's published artifacts (the compiled site and the
release assets). It is not the source repository.

> Note: the `voices-v1` release holds the neural voice models the app downloads
> on first run. It is an internal asset, not an end-user download.
